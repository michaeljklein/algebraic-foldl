newtype KfoldE a b c Source #

The Kleisli arrow of FoldE.

Take an argument and fold over the (fixed) input type until success. To compost, pass the success-result to the next KfoldE and continue folding. (Thus "horizontal" composition.)

It's Category instance represents horizontal composition of FoldE's.
